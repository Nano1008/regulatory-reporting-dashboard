# Financial Transaction Risk Analysis Dashboard

## Overview

This project analyzes a large-scale financial transaction dataset to identify fraud patterns, assess transaction risks, and generate actionable insights for regulatory monitoring.

## Tech Stack

- Python — pandas, numpy

- PostgreSQL — data storage & SQL analysis

- Power BI — dashboard creation and visualization

## Project Workflow
1. Data Cleaning & Preparation: removed invalid records and duplicates

2. SQL Analysis (PostgreSQL)

3. Power BI Dashboard
   
   The dashboard includes:
   
   - Page 1 — Regulatory Overview
  
   - Page 2 — Fraud & Risk Monitoring

   - Page 3 — Transaction Deep Dive
